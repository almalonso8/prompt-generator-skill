# Adobe LLMO Prompt Library Reference

This reference is based on Adobe's official LLM Optimizer Industry Prompt Library. Use these examples as inspiration when generating prompts for different industries.

## Key Principles from Adobe Library

### Branded vs Non-Branded Mix
- **Branded prompts** (35-40%): Include actual company/brand name
- **Non-branded prompts** (60-65%): Generic queries without brand mentions

This mix is critical for GEO tracking - non-branded prompts test if your brand appears in AI responses organically.

### Use Real Brand Names
When generating prompts for a specific company, use their actual name:
- Branded: "What's the baggage limit for Iberia?"
- Branded: "How much does Repsol charge for electricity?"
- Non-branded: "Which airlines have the best baggage policies?"

## How to Use This Reference

When generating prompts for a client:

1. **Identify their industry** (or closest match)
2. **Review relevant topics** from the examples above
3. **Adapt topics** to the client's specific business lines
4. **Mix branded and non-branded** (35-40% branded, 60-65% non-branded)
5. **Use real company names** in branded prompts
6. **Maintain natural language** - prompts should sound like real user queries
